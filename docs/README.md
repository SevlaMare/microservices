# Microservices on backend

> POC microservices utilizing APIs.

Technology: Node with NestJS.

## Quick Guide

### generators

`nest g module users`  
`nest g controller users`  
`nest g service users`

all together  
`nest g resource users`

### new resource

check  
`localhost:3000/users`

route add on:  
/users.service.ts

create interface:  
src/user/interfaces/
